# Army Basketball Player Visualization

This project visualizes Army men's basketball player statistics using **Python**, **Pandas**, and **Pygal**. Player performance metrics are displayed in **radar charts**, allowing for quick comparison of different statistical categories.

## Project Overview

The goal of this project is to explore sports data visualization by transforming raw player statistics into intuitive graphical representations. By using radar charts, users can easily analyze player strengths across multiple performance metrics.

The program reads player data from a CSV file, processes the data using Pandas, and generates a radar chart using Pygal for the selected player.

## Features

- Load and process basketball statistics using **Pandas**
- Generate **radar charts** for player performance visualization
- Select a specific player to view their statistical profile
- Export charts as **SVG files**

## Technologies Used

- Python
- Pandas
- Pygal
- CSV Data Processing

## Example Visualization

Radar charts allow multiple statistical categories to be displayed simultaneously, helping highlight player strengths and weaknesses.
